# 用户阅读钩子

## 你的任务

在文章中自然融入三类阅读钩子，提升打开率、完读率和转发率。

钩子不是生硬插入，而是顺着文章节奏自然放置。像人类编辑写文章——写到某个节点，会本能地放一个悬念、一个吐槽、一个金句。

---

## 钩子类型

### 打开钩子（Open Hook）— 提升打开率

**位置**：H1 标题 + 正文前 3 句

**类型与示例**：

| 类型 | 示例句式 |
|------|---------|
| 信息差 | "你以为 X 是问题，其实 X 是答案。" |
| 反直觉 | "为什么 90% 的人用 X 反而越用越糟？" |
| 数字锚定 | "3 个月，2 次失败，最后靠一个不起眼的细节翻盘。" |
| 痛点直戳 | "你是不是也这样——明明做了一切正确的决定，结果还是不行？" |

**质量标准**：
- 前 3 句不能是背景铺垫、定义解释、客套开场
- 不能是"最近很多人问我..."、"今天聊聊..."这类平淡开头
- 必须让读者在 3 秒内产生"这是什么？我想知道"的冲动

### 留存钩子（Retention Hook）— 提升完读率

**位置**：**每 3-4 段设置 1 个**

**类型与示例**：

| 类型 | 示例句式 |
|------|---------|
| 悬念吊尾 | "但事情没那么简单..." / "接下来发生的事，出乎所有人意料。" |
| 转折预告 | "然而，这才是真正的问题所在。" / "你以为结束了？其实刚开始。" |
| 问题抛给读者 | "你觉得呢？" / "换做是你，你会选哪个？" |
| 未解之谜 | "但真正的坑还在后面，99% 的人都没注意到..." |
| 数据冲击 | "直到我看到一组数据——只有 3% 的人坚持到了最后。" |
| 情绪转折 | "写到这里我有点犹豫，但这事还是得说。" |

**质量标准**：
- 不重复使用同一句式
- 不能是"综上所述"、"总结来说"、"总之"类终结句
- 必须制造"我还想看下一段"的冲动
- 不能与正文内容重复（钩子本身要有信息量）

### 转发钩子（Share Hook）— 提升转发率

**位置**：文章中段（约 1/2-2/3 处）+ 结尾

**类型与示例**：

| 类型 | 示例句式 |
|------|---------|
| 金句 | ≤20 字，精炼、有观点、能独立传播 |
| 身份认同 | "每个认真做 X 的人都懂。" / "经历过的人才知道有多难。" |
| 情绪共鸣 | "说白了就是..." / "这年头，能坚持下来的人都不简单。" |

**质量标准**：
- 每篇至少 1 个可截图转发的金句
- 金句必须短（≤20 字）、有观点、有冲击力
- 转发钩不能与 Step 5.3 的"金句密度"检查重复——转发钩侧重转发动机，金句密度侧重独立传播性
