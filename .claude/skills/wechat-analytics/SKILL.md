# 数据分析与内容策略工作流

## 首次使用安装

如果是第一次使用，需要先安装依赖：

```bash
pip install git+https://github.com/yqdaddy/mamacore.git
claude mcp add mamacore -- mama-server
```

安装完成后重新触发本技能即可。

## 前置检查

确认以下 MCP 工具可用：
- `mama_analyze_account` (账号画像)
- `mama_import_metrics` (数据导入)
- `mama_topic_heatmap` (选题热力图)
- `mama_content_strategy` (内容策略)
- `mama_competitor_gap` (竞品对标)

## 步骤 1: 数据采集

调用 `mama_analyze_account(account_id=账号, days=30)` 获取最近文章数据。
如果 API 不可用，提示用户先用 `mama_import_metrics` 导入历史数据。

## 步骤 2: 指标概览

展示核心指标：
- 文章总数、平均阅读、平均点赞、平均互动率
- 最佳发文时间
- 阅读排行 Top 5

## 步骤 3: 选题热力图

分析哪些话题/框架在你的账号上表现最好。
输出：写了就火的话题 vs 写了就扑的话题。

## 步骤 4: 爆款模式识别

从历史爆款中提取模式：
- 最佳字数
- 最佳发布时间
- 最佳框架
- 标题规律
- 情绪基调

## 步骤 5: 内容策略推荐

综合所有分析数据，输出"下周内容策略报告"：
1. 当前热点中与你账号相关的话题
2. 你还没写过的热门话题
3. 基于历史表现的标题建议
4. 推荐写作框架
5. 最佳发文时间

## 步骤 6: 竞品对比（可选）

如果用户提供竞品账号列表：
调用 `mama_competitor_gap` 进行对比分析。
找出竞品写了但你没写的内容空白。

## 注意事项

- 分析结果需要转化为可执行建议，不只是数据堆砌
- 每个建议都要有"为什么"和"怎么做"
- 保持建议具体：不要写"多写热点"，要写"下周二 21:30 写关于 X 的清单型文章"
